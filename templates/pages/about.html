Welcome to E CAR Website

In particular, this E-CAR website project in Django focuses mainly on dealing with new and used cars. In addition, the system allows for managing customers and car records. To be more precise, the system helps to keep track of customers’ inquiries. Also, the system displays all the available car specifications. Evidently, this project contains an Admin Panel and a customer panel. In an overview of this web application, users can simply register themselves as customers and log into the website. A customer can view, and filter the number of available cars. This specific website is targeted at selling new and pre-owned cars. In fact, it’s kind of like a pre-owned/used car selling system project. The users can search cars by their name or filter cars by their model, year, and price range.
